# @shieldedshell/core

Library for [ShieldedShell](https://github.com/connerkup/shielded-shell): bounded solvers, workspace sandbox, policy engine, and dual-agent orchestration.

```bash
npm install @shieldedshell/core
```

Includes bundled default prompts and OpenHands SDK loop script. Full documentation: [github.com/connerkup/shielded-shell](https://github.com/connerkup/shielded-shell)
