# @shieldedshell/cli

Global CLI for [ShieldedShell](https://github.com/connerkup/shielded-shell): sandboxed agent execution and dual-agent loops.

```bash
npm install -g @shieldedshell/cli@beta
shieldedshell doctor
```

Full documentation: [github.com/connerkup/shielded-shell](https://github.com/connerkup/shielded-shell)
