## 0.1.0 — Public beta (2026-06-16)

### Added

- npm packages `@shieldedshell/cli` and `@shieldedshell/core`
- CLI: `init`, `run`, `shell`, `verify`, `orchestrate`, `loop`, `reconcile`, `doctor`
- Dual-agent loop with ten engine profiles (Claude, Cline, Aider, OpenHands, OpenCode, Copilot, Cursor, Antigravity, OpenClaw)
- Workspace sandbox with overlay, intercept log, and policy engine
- Bounded static solvers and reconciler gate
- Benchmark fixtures in the GitHub repo

### Notes

Beta: APIs and flags may change before 1.0.
